# Laundry Service Landing Page (Responsive)

A responsive landing page for a Laundry Service website built using HTML and CSS. This project demonstrates the use of **Flexbox** and **CSS Media Queries** to create layouts that adapt to desktop, tablet, and mobile devices.

## Features

- Responsive navigation bar
- Hero section with call-to-action button
- Flexbox-based layout
- Tablet and mobile responsive design
- Navigation links hidden on mobile view
- Responsive image and typography

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Media Queries

## Responsive Features

### Desktop
- Horizontal navigation bar
- Hero text and image displayed side by side

### Tablet
- Smaller fonts and images
- Adjusted spacing for better readability

### Mobile
- Navigation links hidden
- Only logo and username displayed
- Hero section stacked vertically
- Responsive button and image sizing

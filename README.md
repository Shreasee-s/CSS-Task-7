#  Laundry Services Landing Page

A simple and responsive landing page for a laundry service website built using **HTML** and **CSS**.

##  Features

- Responsive navigation bar
- Modern UI design
- Hover effects
- Clean layout

##  Technologies Used

- HTML5
- CSS3

##  How to Run

1. Clone this repository:
   ```bash
   git clone <repository-url>
   ```
2. Open the project folder in **Visual Studio Code**.
3. Open `index.html` using the **Live Server** extension.
4. Or Click on : https://shreasee-s.github.io/CSS-Task-5/
